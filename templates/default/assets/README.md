# Assets Directory

Place presentation assets here:
- Images (PNG, JPG, SVG)
- Diagrams
- Data files

## Usage

Reference assets in deck.md using relative paths:

```markdown
![Description](assets/image-name.png)
```

## Tips

- Optimize images before adding (recommended < 1MB each)
- Use descriptive filenames
- PNG for screenshots and graphics
- JPG for photos
- SVG for logos and diagrams (best quality at any size)
