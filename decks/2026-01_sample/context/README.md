# Context Directory

このディレクトリは、AIエージェント(Claude等)にプレゼンテーションのコンテキストを提供するための情報を格納します。

## 使い方

AIと協働してスライドを作成・編集する際、このディレクトリのファイルを参照することで、より適切なサポートが得られます。

## ファイル

- **background.md**: プレゼンの背景、目的、対象聴衆などの情報
- **notes.md**: アイデア、TODO、メモ、参考資料など
- **references/**: 参考資料を格納するディレクトリ(任意)

## ヒント

- プレゼンの目的や対象聴衆を明確に記述する
- 使用したい用語や避けたい表現をリストアップ
- 参考にしたい資料やリンクを保存
- スライドの構成案やアウトラインを下書き

このディレクトリの内容はスライドには含まれませんが、AIがスライドを作成・改善する際の重要な情報源となります。
